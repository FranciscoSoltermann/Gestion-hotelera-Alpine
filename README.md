Hotel Premier - Sistema de Gestión Hotelera
Este proyecto es una aplicación web completa para la gestión de huéspedes, usuarios y reservas de un hotel. Desarrollado con una arquitectura moderna separando el Backend (API REST) del Frontend (Cliente Web).

Tecnologías Utilizadas
Backend:

Java 21

Spring Boot 3.5.0

Spring Data JPA (Hibernate)

Spring Security (BCrypt)

PostgreSQL

Maven

Swagger / OpenAPI

Frontend:

Next.js 14+ (React)

CSS Modules

JavaScript (ES6+)

Prerrequisitos
Antes de ejecutar el proyecto, asegúrate de tener instalado:

Java JDK 21.

Node.js (v18 o superior) y npm.

PostgreSQL (corriendo en la nube).

Maven.
